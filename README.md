# Backend-with-sheryians

Node.js practice projects and exercises from the Sheryians backend workshop.

## Overview

This repository contains small Node.js examples and exercises organized by day. Each folder contains a minimal project you can run locally.

## Project structure

- Day-1/
	- 1-CatMe/ — simple example (`index.js`) with its own `package.json`
- Day-2/ — example project with `index.js` and `package.json`
- Day-3And4/ — combined exercises and a small server in `server.js` and `src/app.js`

## Requirements

- Node.js v14+ (or current LTS)

## How to run

1. Install dependencies (if present):

```
cd <project-folder>
npm install
```

2. Run the example (example for Day-1 CatMe):

```
cd Day-1/1-CatMe
node index.js
```
